# BrainForge North Star

## Mission

Transform raw evidence into trustworthy expertise.

---

## One Command Vision

```bash
brainforge forge \
    --brain <brain-name> \
    --source <source>
```

---

## Expected Result

A complete Brain updated with:

- Evidence Assets
- Evidence Fragments
- Knowledge Units
- Relationships
- Metadata
- Provenance

---

## Success Metric

A user can build a trustworthy Brain from any supported source using one command.
